Unsafe Rust

Unsafe Rust 是 Safe Rust 的超集。

使用 Unsafe Rust 的时候，需要遵守一定的规范，这样可以避免未定义行为的发生。

包含内容如下：

  • 内存
  • 裸指针
  • 安全抽象
  • 联合体
  • FFi
