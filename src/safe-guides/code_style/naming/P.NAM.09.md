## P.NAM.09 定义全局静态变量时需加前缀`G_`和常量有所区分

**【描述】**

为了提升代码可读性和可维护性，有必要将常量的命名和全局静态变量有所区分。所以在定义全局静态变量时，需要以前缀`G_`命名。



**【反例】**

```rust
static EVENT: [i32;5]=[1,2,3,4,5];
const MAGIC_NUM: i32 = 65 ;
```

**【正例】**

```rust
static G_EVENT: [i32;5]=[1,2,3,4,5];
const MAGIC_NUM: i32 = 65 ;
```
